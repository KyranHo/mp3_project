## YouTube to MP3 Converter

Utilises the following:
- pytube-fix
- ffmpeg

Uses pytube-fix to convert from YouTube URL into a M4A file format, which is later converted to MP3 using ffmpeg.

Do install ffmpeg and replace the file path accordingly in the "conversion" function. 

This project was made for the sole purpose of getting an offline playlist for overseas travel where roaming data may be limited.
